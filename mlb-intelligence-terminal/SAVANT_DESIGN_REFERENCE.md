# Savant-Style Dashboard Design Reference

## Key Visual Elements

### Header & Navigation
- Clean white header with navy background
- MLB Statcast branding on left
- Horizontal navigation: Player Search, Leaderboards, Statcast 101, About
- Search bar on right with player search functionality
- Left sidebar navigation for main sections

### Player Profile Page (Page 1)
- Player photo on left (professional headshot)
- Player name and number prominently displayed
- Team logo and basic info (position, handedness, age, height, weight)
- Key stats displayed in large text: HR, AVG, OPS with percentile indicators
- Season stats table on right showing PA, R, H, 2B, 3B, HR, RBI, BB%, K%, wOBA, xwOBA
- Horizontal tab navigation: Overview, Statcast, Splits, Game Log, Zone Charts, Spray Chart, Pitch Selection, News
- Three-column layout below tabs:
  1. **Statcast Percentile Rankings** - Horizontal bar charts showing percentile (0-100) with color coding (blue=poor, red=great)
  2. **Spray Chart** - Baseball field visualization with hit locations
  3. **Pitch Velocity** - Line graph showing velocity distribution by pitch type

### Prospect Page (Page 2)
- Left sidebar with navigation menu (Overview, Stats, Game Log, Reports, Development, News, Watch, Transactions)
- Player photo and basic info
- Prospect rank badge (e.g., "#5 IN ORGANIZATION")
- Level progression timeline showing development path (LOW-A → HIGH-A → DOUBLE-A → TRIPLE-A → MLB)
- Season stats table
- Two-column lower section:
  1. **Statcast vs. Level** - Multi-line chart comparing player to level average across metrics
  2. **Scouting Report** - Tool grades (Hit, Power, Run, Arm, Field) with numeric scores and visual bar indicators

### Leaderboard Page (Page 3)
- Left sidebar with main navigation
- Filter options: Season, Position, Team dropdowns
- Customize Columns button
- Tabs: Hitting, Pitching, Fielding
- Data table with columns: Rank, Player, Team, PA, AVG, wOBA, xwOBA, EV (MPH), Hard Hit %, Barrel %, xBA, xSLG, BB %, K %, Chase %, WAR
- Player photos in table
- Team logos in table
- Pagination at bottom

## Color Scheme
- **Navy Blue**: Primary background (#001F3F or similar)
- **White**: Primary text and backgrounds
- **Light Gray**: Secondary text and borders
- **Red**: Elite/Great performance indicators
- **Blue**: Poor/Below-average indicators
- **Green**: Positive indicators
- **Light Blue/Cyan**: Neutral/Average indicators

## Typography
- Bold, clean sans-serif for headers
- Regular sans-serif for body text
- Large, prominent numbers for key stats
- Percentile indicators in smaller text below stats

## Layout Principles
- Clean, organized grid layouts
- Ample whitespace
- Clear visual hierarchy
- Professional, data-focused design
- Minimal decoration/embellishment
- Focus on information density without clutter

## Key Components
1. **Percentile Bars**: Horizontal bars from 0-100 with color gradient
2. **Spray Charts**: Baseball field visualization with hit markers
3. **Line Charts**: Multi-line trend charts for metric comparison
4. **Data Tables**: Clean, sortable tables with team logos and player photos
5. **Tool Grade Cards**: Numeric scores with visual bar indicators
6. **Timeline Progression**: Visual representation of player development path
7. **Stat Boxes**: Large numeric displays with labels and context

## Information Architecture
- Player profiles as primary focus
- Quick access to key metrics and percentiles
- Detailed statistical breakdowns available via tabs
- Comparison tools (vs. level, vs. league)
- Professional scouting integration
- Clean navigation between sections
